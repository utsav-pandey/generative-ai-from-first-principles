# Generative AI from First Principles

A comprehensive repository for learning Generative AI from the ground up.

This repository focuses on developing a rigorous understanding of modern Generative AI by combining mathematical intuition, probabilistic modelling, theoretical foundations, handwritten derivations, PyTorch implementations, and experimental analysis.

The repository is primarily based on the **NPTEL course "Generative AI" by Prof. Prathosh A.P. (IISc Bangalore)**, along with standard textbooks and research papers.

---

## Philosophy

Generative AI is built upon probability theory, optimization, deep learning, and modern transformer architectures.

This repository follows a **first-principles approach**, where every topic is studied by:

- Developing mathematical intuition.
- Understanding the probabilistic formulation.
- Studying important derivations.
- Learning the underlying theory.
- Implementing every model in PyTorch.
- Reproducing important experiments.
- Documenting observations through professional LaTeX reports.

The objective is not only to use Generative AI models but also to understand the mathematics and principles behind them.

---

## Objectives

- Build a strong foundation in Generative AI.
- Understand probabilistic deep generative models.
- Learn the mathematics behind modern generative models.
- Implement every major model in PyTorch.
- Study the evolution of Generative AI from GANs to Large Language Models.
- Document experiments through professional LaTeX reports.
